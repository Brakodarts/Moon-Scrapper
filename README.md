# MoonScrapper

A modular resource-management game engine built in C# and .NET, inspired by city-building and survival games .

## 🚀 Features
- **Real-Time Game Loop:** Resources are generated every second in a background tick system.
- **Interactive Console UI:** Built with `Spectre.Console` for a modern and colorful terminal experience.
- **MVC Architecture:** Clean separation between Game Models, Logic Services, and the UI.

## 🛠️ Tech Stack
- **Language:** C# / .NET
- **UI Library:** Spectre.Console
- **Architecture:** Model-View-Controller (MVC)

## 🎮 How to Play
1. Clone the repository.
2. Run `dotnet run` in your terminal.
3. Watch your resources grow!
4. Press the B key to open the Building section.
